Linux Looks and Usability

Now, you know that in terms of looks, Linux is equal to if not better than Windows. But looks are not the only thing you want to check if you buy something. Ohh, but it looks like Linux is free and open source. Now, what the heck is open source? It means that its code is available to see and download for free. Also, you can change it and distribute it for free.


Security

Next comes security. You might be thinking that if the code is available for everyone to see, someone can check the vulnerabilities and alter it to hack or corrupt a system. But here's an amazing thing. If one person with malicious intent is trying to find loopholes, hundreds, if not thousands, are trying to find it so that they can be fixed. Thus, Linux is way more safe than Windows in a straightforward comparison. Also, you can be sure that the creators are not spying on you because if they were, someone would find out in their code and tell the world.

But there is always a catch !

Now I think I gave you enough reasons to use Linux. But there is always a catch to good things. In this case, the catch is installing applications. To install applications, you have to use the provided package manager, which I think contains all the packages a non-programmer needs. Also, they are all free to use. But for beginner programmers, some Linux distributions may be quite punishing. So, I have curated the below list ranking some Linux distros or you can say flavors on the basis of their learning curve.


Some distributions you can use

Beginner-friendly:

  • Ubuntu: Known for its user-friendly interface and large community support.
  • Manjaro: Offers a more user-friendly experience compared to Arch Linux, while still providing a lot of flexibility.
  • Mint: Designed with a focus on ease of use and a familiar interface.
  • Pop!_OS: Developed by System76, designed to be user-friendly and optimized for hardware support.
  • Fedora: User-friendly, but slightly more focused on advanced features and updates than Ubuntu.

Intermediate (Little punishing):

  • Arch Linux: Requires manual installation and configuration, making it more suitable for users with some Linux experience.
  • Kali Linux: Primarily used for penetration testing and security assessments, requires a good understanding of Linux.
  • Garuda Linux: Based on Arch Linux and offers a variety of desktop environments. It inherits some complexity from Arch.

Advanced (Hell):

  • NixOS: Known for its unique package management system and configuration approach. It can be quite challenging, especially for those not familiar with Nix.
  • Gentoo: A highly customizable source-based distribution that requires extensive manual configuration and compilation.
  • Slackware: One of the oldest Linux distributions, known for its minimal hand-holding and manual configuration.
  • LFS (Linux From Scratch): This isn't a traditional distribution but a book that guides users in building a custom Linux system from source code. It's the ultimate challenge for learning Linux.


Now I know the above list can be quite intimidating at first, so if you are a little bit experienced in using Linux, you can choose whatever you want. But for beginners or a little above beginners, I will recommend Ubuntu or Manjaro. I have used them myself and have faced a little difficulties at first. But once you get a hang of it, I assure you, you will only get back to Windows for gaming or some software which is only supported by Windows.
