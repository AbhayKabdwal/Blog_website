Regular Expressions in Python: Basics to Advanced

1. Introduction to Regular Expressions

1.1 What are Regular Expressions?

Regular Expressions (regex or regexp) are sequences of characters that define a search pattern. They enable string manipulation based on patterns.

1.2 Why Use Regular Expressions?

Regular expressions provide a powerful and flexible way to search, match, and manipulate strings. They are widely used for tasks like validation, extraction, and search-and-replace operations.

2. Installing the re Module

pip install regex

3. Getting Started with Regular Expressions in Python

3.1 Importing the re Module

import re

3.2 Basic Patterns: Matching Literal Strings

        
pattern = r"hello"
text = "Hello, world!"
match = re.search(pattern, text)
print(match.group())  # Output: "hello"

4. Understanding Patterns

4.1 Metacharacters: Building Blocks of Patterns

        
pattern = r"he.l."
text = "hello"
match = re.search(pattern, text)
print(match.group())  # Output: "hello"

4.2 Character Classes and Quantifiers

        
pattern = r"\d{3}-\d{2}-\d{4}"
text = "Social Security Number: 123-45-6789"
match = re.search(pattern, text)
print(match.group())  # Output: "123-45-6789"

5. Searching for Patterns

5.1 The re.search() Function

        
pattern = r"world"
text = "Hello, world!"
match = re.search(pattern, text)
print(match.group())  # Output: "world"

5.2 Matching at the Beginning with re.match()

        
pattern = r"Hello"
text = "Hello, world!"
match = re.match(pattern, text)
print(match.group())  # Output: "Hello"

5.3 Finding All Occurrences with re.findall()

        
pattern = r"\b\w+\b"
text = "This is a simple sentence."
matches = re.findall(pattern, text)
print(matches)  # Output: ['This', 'is', 'a', 'simple', 'sentence']

6. Grouping and Capturing

6.1 Creating Groups with Parentheses

        
pattern = r"(\d{3})-(\d{2})-(\d{4})"
text = "Date of Birth: 123-45-6789"
match = re.search(pattern, text)
print(match.group(1))  # Output: "123"
print(match.group(2))  # Output: "45"
print(match.group(3))  # Output: "6789"

6.2 Capturing Matched Text

        
pattern = r"hello"
text = "Hello, world!"
match = re.search(pattern, text)
print(match.group())  # Output: "hello"

6.3 Non-Capturing Groups

        
pattern = r"Mr(?:s|)\. Smith"
text = "Mr. Smith went to the store."
match = re.search(pattern, text)
print(match.group())  # Output: "Mr. Smith"

7. Modifiers and Special Characters

7.1 Dot (.) as Any Character

        
pattern = r"Hello....world"
text = "Hello, Python world!"
match = re.search(pattern, text)
print(match.group())  # Output: "Hello, Python world"

7.2 Asterisk (*) for Zero or More Occurrences

        
pattern = r"ab*c"
text = "ac, abc, abbc, abbbc"
matches = re.findall(pattern, text)
print(matches)  # Output: ['ac', 'abc', 'abbc', 'abbbc']

7.3 Plus (+) for One or More Occurrences

        
pattern = r"ab+c"
text = "ac, abc, abbc, abbbc"
matches = re.findall(pattern, text)
print(matches)  # Output: ['abc', 'abbc', 'abbbc']

7.4 Question Mark (?) for Zero or One Occurrence

        
pattern = r"ab?c"
text = "ac, abc, abbc, abbbc"
matches = re.findall(pattern, text)
print(matches)  # Output: ['ac', 'abc']

8. Advanced Techniques

8.1 Greedy vs. Non-Greedy Matching

        
pattern = r"Hello.*world"
text = "Hello, Python world!"
match = re.search(pattern, text)
print(match.group())  # Output: "Hello, Python world"

pattern = r"Hello.*?world"
text = "Hello, Python world!"
match = re.search(pattern, text)
print(match.group())  # Output: "Hello, world"

8.2 Lookahead and Lookbehind Assertions

        
pattern = r"\b(?=\w{5}\b)\w+\b"
text = "Match words with exactly 5 characters"
matches = re.findall(pattern, text)
print(matches)  # Output: ['Match', 'words']

8.3 Using Backreferences

        
pattern = r"(\w+) \1"
text = "Repeat Repeat Hello Hello World"
matches = re.findall(pattern, text)
print(matches)  # Output: ['Repeat', 'Hello']

9. Common Use Cases

Validation: Validate input, like email addresses, phone numbers, or passwords.

Extraction: Extract specific information from a text, like dates, URLs, or numbers.

Search and Replace: Find and replace text based on patterns.

Tokenization: Split text into tokens (words, sentences) based on specific criteria.
