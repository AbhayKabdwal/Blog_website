C++ Basics: Variables, Data Types, and Operators

Variables

Variables are used to store and manipulate data in a program.

Declaration

// Syntax: data_type variable_name;
    int age;

Initialization

// Syntax: data_type variable_name = initial_value;
    int age = 25;

Data Types

C++ supports various data types for different kinds of values.

Fundamental Data Types

int: Integer data type

int number = 10;

float: Floating-point data type

float pi = 3.14;

double: Double-precision floating-point data type

double price = 99.99;

char: Character data type

char grade = 'A';

Derived Data Types

Array: Collection of elements of the same data type

int numbers[5] = {1, 2, 3, 4, 5};

Pointer: Stores memory addresses

int* ptr = &number;

Operators

Operators perform operations on variables and values.

Arithmetic Operators

Addition:

int sum = 5 + 3;

Subtraction:

int difference = 8 - 2;

Multiplication:

int product = 4 * 6;

Division:

float quotient = 10.0 / 2;

Modulus (Remainder):

int remainder = 15 % 7;

Relational Operators

Equal to:

bool isEqual = (a == b);

Not equal to:

bool isNotEqual = (a != b);

Greater than:

bool isGreater = (a > b);

Less than:

bool isLess = (a < b);

Logical Operators

AND:

bool andResult = (x && y);

OR:

bool orResult = (x || y);

NOT:

bool notResult = !x;

Assignment Operators

Assignment:

int a = 10;

Add and Assign:

Subtract and Assign:

Multiply and Assign:

a *= 2;  // Equivalent to a = a * 2;

Divide and Assign:

Modulus and Assign:

Increment and Decrement Operators

b--;  // Equivalent to b = b - 1;

If-Else Statements

Introduction

In C++, the if and else statements are used for conditional execution of code. They allow you to control the flow of your program based on certain conditions.

Syntax

Basic If Statement

if (condition) {
    // Code to be executed if the condition is true
}
Example:
#include <iostream>
int main() {
    int x = 10;
    if (x > 5) {
        std::cout << "x is greater than 5." << std::endl;
    }
return 0;
}

If-Else Statement

if (condition) {
    // Code to be executed if the condition is true
} else {
    // Code to be executed if the condition is false
}
Example:
#include <iostream>
int main() {
    int x = 3;

    if (x > 5) {
        std::cout << "x is greater than 5." << std::endl;
    } else {
        std::cout << "x is not greater than 5." << std::endl;
    }
return 0;
}

If-Else If-Else Statement

if (condition1) {
    // Code to be executed if condition1 is true
} else if (condition2) {
    // Code to be executed if condition2 is true
} else {
    // Code to be executed if none of the conditions are true
}
Example:
#include <iostream>
int main() {
    int x = 3;

    if (x > 5) {
        std::cout << "x is greater than 5." << std::endl;
    } else if (x == 5) {
        std::cout << "x is equal to 5." << std::endl;
    } else {
        std::cout << "x is less than 5." << std::endl;
    }
return 0;
}

Basics of Loops in C++

Loops in C++ are used to repeatedly execute a block of code until a certain condition is met. There are three main types of loops in C++:

1. for loop:

The for loop is used when the number of iterations is known.

for (initialization; condition; increment/decrement) {
    // Code to be repeated
}

Example:

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 5; ++i) {
        cout << i << " ";
    }
    return 0;
}

2. while loop:

The while loop is used when the number of iterations is not known in advance.

while (condition) {
    // Code to be repeated
}

Example:

#include <iostream>
using namespace std;

int main() {
    int count = 0;
    while (count < 3) {
        cout << count << " ";
        count++;
    }
    return 0;
}

3. do-while loop:

The do-while loop is similar to the while loop, but it ensures that the code block is executed at least once, as the condition is checked after the loop body.

do {
    // Code to be repeated
} while (condition);

Example:

#include <iostream>
using namespace std;

int main() {
    int num = 5;
    do {
        cout << num << " ";
        num--;
    } while (num > 0);
    return 0;
}

Basics of Functions in C++

Functions are a fundamental building block in C++ programming, allowing for code organization, reusability, and abstraction. Here are the key aspects of functions:

Function Declaration

// Example: Function Declaration
int add(int a, int b);

Function Definition

// Example: Function Definition
int add(int a, int b) {
    return a + b;
}

Function Call

// Example: Function Call
int result = add(5, 3);

Return Type

// Example: Function with Void Return Type
void displayMessage() {
    cout << "Hello, World!" << endl;
}

Parameters

// Example: Function with Parameters
double calculateAverage(double num1, double num2) {
    return (num1 + num2) / 2;
}

Default Arguments

// Example: Function with Default Argument
int multiply(int a, int b = 2) {
    return a * b;
}

Function Overloading

// Example: Function Overloading
int add(int a, int b);
double add(double a, double b);

Recursion

// Example: Recursive Function
int factorial(int n) {
    if (n == 0 || n == 1)
        return 1;
    else
        return n * factorial(n - 1);
}

Scope of Variables

// Example: Local Scope
void exampleFunction() {
    int localVar = 10;
    // localVar is only accessible within this function
}

Function Prototypes

// Example: Function Prototype
int multiply(int a, int b);

Memory Management in C++

Memory management is a critical aspect of C++ programming, involving the allocation and deallocation of memory for variables and data structures. Here are the basics:

1. Stack and Heap Memory

Stack Memory

  • Description: Stack memory is used for local variables and function call management.
  • Allocation: Automatically allocated and deallocated during function calls.

Example:

void exampleFunction() {
    int x = 10;  // x is allocated on the stack
    // ...
}

Heap Memory

  • Description: Heap memory is used for dynamic memory allocation, allowing objects to persist beyond their scope.
  • Allocation: Manually allocated and deallocated using new and delete or malloc and free.

Example:

int* dynamicVar = new int;  // Allocate integer on the heap
// ...
delete dynamicVar;  // Deallocate memory

2. Pointers

Pointers

  • Description: Pointers store memory addresses, allowing manipulation of memory directly.
  • Declaration: int* ptr; declares a pointer to an integer.

Example:

int x = 5;
int* ptr = &x;  // ptr holds the address of x

Dereferencing

  • Description: Access the value stored at a memory address using the dereference operator *.

Example:

int y = *ptr;  // y now holds the value stored at the address pointed by ptr

3. Memory Leak

Description

  • Description: Occurs when memory is allocated but not deallocated, leading to a gradual depletion of available memory.

Example:

void memoryLeakExample() {
    int* dynamicVar = new int;
    // ... // No deallocation, causing a memory leak
}
