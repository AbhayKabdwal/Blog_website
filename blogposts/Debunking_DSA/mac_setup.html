Setting Up C++ Development Environment with Visual Studio Code on macOS

Prerequisites

Make sure you have the following installed on your macOS machine:

  • Visual Studio Code
  • Xcode Command Line Tools

1. Install Homebrew (Optional)

If you don't have Homebrew installed, you can install it using the following command in the terminal:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2. Install GCC (GNU Compiler Collection)

Use Homebrew to install GCC:

brew install gcc

This step is optional as macOS comes with a version of Clang, but using GCC can be beneficial in some cases.

3. Install Visual Studio Code

Download and install Visual Studio Code from the official website:

Visual Studio Code Download

4. Install Extensions

Open Visual Studio Code, and install the following extensions for C++ development:

  • C/C++ Extension

5. Create a C++ File

Create a new folder for your C++ projects and open it in Visual Studio Code. Inside the folder, create a new file named main.cpp with the following content:

#include <iostream>

int main() {
    std::cout << "Hello, World!" <<  std::endl;
    return 0;
}

7. Build and Run

  • Open your main.cpp file in Visual Studio Code.
  • Press Cmd + Shift + B to build your code.
  • Press F5 to run your code.
