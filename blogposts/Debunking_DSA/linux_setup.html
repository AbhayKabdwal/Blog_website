Ubuntu

1. Install Visual Studio Code

  • Go to Visual Studio Code website.
  • Download and install the .deb package.

2. Install C++ Extension

  • Open VS Code.
  • Navigate to Extensions (or press Ctrl + Shift + X).
  • Search for "C/C++" by Microsoft and install it.

3. Install Build Essentials

sudo apt-get update
sudo apt-get install build-essential

4. Install C++ Compiler (g++)

sudo apt-get install g++

5. Verify Installation

g++ --version

Verify that the version information is displayed.

6. Create a Hello World Program

  • Open VS Code.
  • Create a new file, e.g., hello.cpp.
  • Write a simple C++ program.
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

7. Build and Run

  • Save the file.
  • Open the terminal in VS Code.
  • Run the following commands:
g++ -o hello hello.cpp
./hello

Verify that "Hello, World!" is printed.
