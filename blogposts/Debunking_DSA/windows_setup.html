Setting Up C++ Development Environment with Visual Studio Code on Windows

Prerequisites

Before setting up your C++ development environment, ensure that you have the following installed on your Windows machine:

  • Visual Studio Code
  • MinGW-w64 (Minimalist GNU for Windows) for GCC compiler
  • CMake (optional, for CMake-based projects)

Steps

1. Install MinGW-w64

Download and install MinGW-w64 by following these steps:

  1. Visit the MinGW-w64 download page and download the installer.
  2. Run the installer and select the following options:
    • Version: Latest
    • Architecture: x86_64 (for 64-bit system) or i686 (for 32-bit system)
  3. Set the installation directory (e.g., C:\).
  4. Add the MinGW-w64 bin directory to your system PATH. This is usually C:\mingw64\bin or C:\mingw32\bin depending on your architecture.

2. Verify GCC Installation

Open a command prompt and run the following commands to verify that GCC is installed correctly:

gcc --version
g++ --version

3. Install Visual Studio Code

Download and install Visual Studio Code from official website. Follow the installation instructions provided.

4. Install the C/C++ Extension

Open Visual Studio Code, go to the Extensions view by clicking on the Extensions icon in the Activity Bar on the side of the window or use the shortcut Ctrl + Shift + X.

Install the following extensions:

  • C/C++ -- by microsoft
  • Code Runned -- by Jun Han

5. Run Test program

Create a new folder for your C++ projects and open it in Visual Studio Code. Inside the folder, create a new file named main.cpp with the following content:

#include <iostream>

int main() {
std::cout << "Hello, World!" << std::endl;
return 0;
}

Now click on the triangle icon on the top right.

If the code runs successfully you will get a Hello, World! written on the output section.

If not check if you completed all the steps correctly as described above.
