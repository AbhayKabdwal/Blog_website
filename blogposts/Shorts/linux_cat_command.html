`cat` Command Usage in Linux

Basic Usage

Display the contents of a file

cat file_name

File name can be either relative or absolute.

Display Multiple files

cat file1 file2

Create a New File

cat > file_name

Append to a File

cat >> file_name

Display Number of Lines

cat -v file_name
