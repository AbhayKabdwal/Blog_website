`ls` Command Usage in Linux

Basic Usage

List the contents of the specified directory.

ls directory_name

Directory name can be either relative or absolute.

List wit Details

ls -l

List including hidden files

ls -a

Sort by Modification Time

ls -lt

List along with Human-Readable Sile Sizes

cd ..
