`mv` Command Usage in Linux

Basic Usage

Move Files or Directories

mv file_name destination

Both filename and directory name can be either relative or absolute.

Renaming files or Directories

mv old_name new_name

Moving multiple files

mv file1 file2 directory/

Force Move (Override Existing Files)

mv -f source_directory destination_directory
The -f option forces the move, overriding existing files without asking for confirmation. Forcefully moves file.txt to the /path/to/destination/ directory, replacing any existing file with the same name.
