`cd` Command Usage in Linux

Basic Usage

Change the current directory to the specified directory.

cd directory_name

Directory name can be either relative or absolute.

Move to Home Directory

cd

Move to Root Directory

cd ~

Move to Previous Directory

cd -

Move Up One Level

cd ..

Change the current directory using a variable.

MY_DIR="Documents"
cd $MY_DIR
