What is the Shell?

The shell is a program that takes keyboard commands and passes them to the operating system to carry out. It's the command line interface that allows users to interact with the operating system.

Terminal Emulators

To access the shell, we need a terminal emulator. Popular terminal emulators include:

  • Konsole (KDE)
  • GNOME Terminal (GNOME)
  • Other terminal emulators (e.g., xterm, rxvt)

First Keystrokes

Launch the terminal emulator

See the shell prompt: 

[me@linuxbox ~]$

Type gibberish to see an error message

Use the up-arrow key to recall previous commands (command history)

Use the left and right arrow keys to move the cursor

A Few Words About Mice and Focus

Use the mouse to copy and paste text in the terminal emulator

Highlight text by holding down the left mouse button and dragging

Press the middle mouse button to paste text

Note: CTRL-C and CTRL-V don't work for copy and paste in the terminal

Try Some Simple Commands

date
: displays the current time and date

cal
: displays a calendar of the current month

df
: displays the current amount of free space on disk drives

free
: displays the amount of free memory

Ending a Terminal Session

Close the terminal emulator window

Enter the 

exit
command at the shell prompt
