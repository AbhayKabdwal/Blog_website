Chapter 4: Manipulating Files and Directories

Introduction

At this point, we are ready for some real work! This chapter will introduce the following commands:

  • cp—Copy files and directories.
  • mv—Move/rename files and directories.
  • mkdir—Create directories.
  • rm—Remove files and directories.
  • ln—Create hard and symbolic links.

These five commands are among the most frequently used Linux commands. They are used for manipulating both files and directories.

While some tasks are easier with a graphical file manager, command-line programs offer power and flexibility. For instance, copying all HTML files from one directory to another, but only those that do not exist in the destination directory or are newer, is simpler with the command line:

cp -u *.html destination

Wildcards

The shell provides special characters called wildcards to help rapidly specify groups of filenames. These wildcards are also known as globbing and allow you to select filenames based on patterns of characters.

Wildcard Matches
* Any characters
? Any single character
[characters] Any character that is a member of the set characters
[!characters] Any character that is not a member of the set characters
[[:class:]] Any character that is a member of the specified class

Commonly used character classes:

Character Class Matches
[:alnum:] Any alphanumeric character
[:alpha:] Any alphabetic character
[:digit:] Any numeral
[:lower:] Any lowercase letter
[:upper:] Any uppercase letter

Commands

mkdir—Create Directories

The mkdir command is used to create directories:

mkdir directory...

Example:

mkdir dir1 dir2

This creates two directories, dir1 and dir2.

cp—Copy Files and Directories

The cp command copies files or directories:

cp item1 item2

Example:

cp file1 file2

This copies file1 to file2. If file2 exists, it is overwritten.

mv—Move and Rename Files

The mv command is used to move or rename files:

mv item1 item2

Example:

mv file1 file2

This moves or renames file1 to file2.

rm—Remove Files and Directories

The rm command deletes files and directories:

rm item...

Be careful with rm as it does not have an undelete command. Use wildcards cautiously.

ln—Create Links

The ln command creates hard and symbolic links:

ln file link

Example:

ln fun fun-hard

This creates a hard link to fun named fun-hard.

Examples and Exercises

Creating Directories

To create a directory named playground, use:

mkdir playground

Copying Files

To copy a file from /etc to the current directory:

cp /etc/passwd .

Moving and Renaming Files

To rename a file:

mv passwd fun

Creating Hard Links

To create hard links:

ln fun fun-hard

Creating Symbolic Links

To create symbolic links:

ln -s fun fun-sym
